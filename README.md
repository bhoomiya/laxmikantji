# laxmikantji

Personal portfolio website.

**Overview**

- **Project type:** Static personal/portfolio site
- **Purpose:** Showcase profile, work, insights, and contact information

**Included files**

- `index.html` — Home / landing page
- `profile.html` — About / resume
- `work.html` — Projects and work samples
- `insights.html` — Blog posts or insights
- `contact.html` — Contact form / details
- `style.css` — Site styles
- `main.js` — Interactive scripts

**How to view**

1. Open `index.html` in any modern browser, or
2. Serve the folder locally (recommended) and visit http://localhost:8000

	 - Using Python 3:

		 ```bash
		 python -m http.server 8000
		 ```

**Development**

- Edit the HTML/CSS/JS files and refresh the browser to see changes.
